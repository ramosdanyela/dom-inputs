Exercícios sobre INPUTS e Manipulação

Exercício 1

Crie um input do tipo texto e um botão. Ao clicar no botão, o texto que está dentro do input deve ser colocado dentro de uma tag p.



Exercício 2

Crie dois inputs do tipo number e um botão. Ao clicar no botão, o valor dos dois inputs deve ser somado e o resultado deve ser colocado dentro de uma tag p.

+

Exercício 3

Crie um input do tipo texto e um botão. Ao clicar no botão, o texto que está dentro do input deve ser colocado dentro de uma tag li. Se o input estiver vazio e o botão ser clicado, um alerta deve ser mostrado na tela dizendo: Por favor, insira um texto.



    Exercício 4

    Crie um select com 3 options. Ao selecionar uma das options, o valor selecionado deve ser colocado dentro de uma tag h3.

    Exercício 5

    Crie 3 checkbox com os valores: "Ocupado", "Livre" e "Indisponível". Ao marcar um dos checkbox, o valor do checkbox marcado deve ser colocado dentro de uma tag p.

    Bonus: Se mais de um checkbox estiver marcado, crie outras tags p para cada valor marcado.

    Bonus 2: Ao desmarcar uma checkbox a tag p deve ser retirada da tela.





    Exercício 6

    Crie 3 radio buttons com os valores: "Masculino", "Feminino" e "Outro". Ao selecionar um dos radio buttons, o valor selecionado deve ser colocado dentro de uma tag p.

    HTML

    DIV EXERCISE CONTAINER:

    INPUTS

    class = 'input'
    General syntax = inputXXNn, where:

    XX = typefield
      Typefields
    • TF = textfield
    • NF = numberfield
    • SLC = select
    • CB = checkbox
    • RD = radio
    N = exercise number
    n = inputnumber in exercise

    example:
    An input that is in exercise 3, is the type radio e there are 4 options, it would be like that:
    inputRD31
    inputRD32
    inputRD33
    inputRD34
